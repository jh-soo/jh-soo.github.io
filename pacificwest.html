Information Architecture

Home & Product page

Home page displays banner to show events and promotions. Home page also offers a simple overview of all products offered by Pacific West.

Product page displays product information and cooking instructions. User can navigate to nearby stores to make purchases, it can be replaced with e-commerce functions in the next stage.

Recipe page

Recipe page displays to 3 sections: latest recipe, recipe from invited chef and recipe create by users.

Recipe page detailed into ingredients, upload video/picture steps, user ratings and review.

Points and Reward

Users can earn points from engage in app, such as spinning the lucky wheel, participating in competitions, posting recipes and posting comments.

Accumulated points can be use to exchange voucher from selected partners.
